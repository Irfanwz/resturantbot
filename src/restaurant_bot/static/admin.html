How to create a Telegram bot (2 minutes)

  1. Open Telegram → Search @BotFather
  2. Send /newbot → Enter a name → Enter a username
  3. Copy the Bot Token BotFather gives you
  4. Paste it below → Save → Click Auto-Setup Webhook
